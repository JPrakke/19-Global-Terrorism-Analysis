Data Analysis

Where is the data for 1993?

The original PGIS data, whereupon the 1970-1997 GTD information are based, consisted of hard-copy index cards originally. When being electronically transferred by researchers, significant data for 1993 went missing and is omitted due to this.

Database Debacles

Data type for loading in MySQL, INT and DOUBLE cannot take negative. "Solution: INT Signed, DOUBLE signed" But later we did not end up using this as MySQL Data Import would give a lot of issues, and we changed the approach to using Pandas to load the data.

Cleaning up the Data

We decided to only take data from the current century (2000-2016) to help performance. Entries that were missing important tools such as lattitude and longitude were removed for mapping purposes. After cleaning with PANDAS, the data was loaded back into mySQL with 99,801 rows.

Checked for Accuracy

For the integrity of our information, we cross-referenced our datasets by querying major terrorism events (September 11th, Pulse Nightclub, Etc.) from our database, and comparing our outcomes to official news reports to match correct values.
